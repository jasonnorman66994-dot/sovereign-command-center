# Video Production: Azure Architecture in Action

Version: 1.0
Last Updated: 2026-04-16
Duration Target: 40–45 seconds
Platform: LinkedIn

---

## Script Outline

### Opening (5–7 s)

- **Visual:** Title card "Azure Architecture in Action" with gradient background (blue → green).
- **Voiceover:** "Cloud strategy isn't one-size-fits-all. Let's explore three proven Azure architectures."
- **On-screen text:** Subtitle — "Cloud strategy isn't one-size-fits-all."
- **Motion:** Fade-in of title, subtle animated cloud icon.

### Segment 1 — Secure E-Commerce (10 s)

- **Visual:** Shopping cart + shield icon, blue theme.
- **Voiceover:** "For high-traffic and sensitive data, Secure E-Commerce ensures PCI compliance and performance with WAF and Managed SQL."
- **On-screen text:** "Secure E-Commerce" + bullet points:
  - WAF
  - Managed SQL
  - 99.9% uptime
- **Motion:** Icon slides in from left; bullet points fade in one by one.

### Segment 2 — Enterprise Hybrid (10 s)

- **Visual:** Building + cloud icon, orange theme.
- **Voiceover:** "Hybrid Integration bridges on-prem systems with Azure using VPN or ExpressRoute, layered security, and hybrid identity."
- **On-screen text:** "Enterprise Hybrid" + bullet points:
  - VPN / ExpressRoute
  - NSGs
  - Hybrid Identity
- **Motion:** Icon zooms in; bullet points appear with slight bounce.

### Segment 3 — Global SaaS (10 s)

- **Visual:** Globe + cloud icon, green theme.
- **Voiceover:** "Global SaaS delivers worldwide scalability with Traffic Manager, tenant isolation, and geo-replication."
- **On-screen text:** "Global SaaS" + bullet points:
  - Multi-Region VNets
  - Traffic Manager
  - Geo-replication
- **Motion:** Globe rotates slowly; bullet points fade in.

### Closing (5–8 s)

- **Visual:** All three icons side-by-side on a dark gradient background.
- **Voiceover:** "Explore the full toolkit on GitHub and choose the right path for your cloud strategy."
- **On-screen text:** "Choose the Right Path for Your Cloud Strategy"
- **Footer text:** "github.com/JasonNorman/Azure-Networking-Docs"
- **Motion:** Icons pulse slightly; footer link slides up.
